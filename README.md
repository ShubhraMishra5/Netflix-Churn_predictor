# Netflix-Churn-predictor
An interactive R Shiny web application that uses Machine Learning to predict whether a Netflix user will stay or cancel their subscription.

# Key Features
Predictive Modeling: Uses a Random Forest algorithm to analyze viewing habits, login frequency, and account profiles.

Netflix UI: A custom-themed interface built with bslib and CSS to mirror the real Netflix dashboard.

Reactive Feedback: Displays dynamic messages and memes based on the prediction outcome.

Clean Architecture: Organized with a dedicated www/ folder for optimized asset management.
